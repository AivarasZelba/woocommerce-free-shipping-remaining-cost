This plugin shows how much money user should spend in order to get free shipping if available.

== Description ==

Encourage your customers to spend more, by notifiying them how much more they need to buy, in order to qualify for Free Shipping.

This plugin requires Woocommerce atleast 2.0

== Installation ==

Upload the Woocommerce Free Shipping Notification plugin to your website.
Go to Woocommerce > Settings > Shipping > Free Shipping and write sum of money into Minimum Order Amount field.
